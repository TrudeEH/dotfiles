# How to start VR
1. Launch SlimeVR if needed
2. Launch WiVRn
3. Launch Wayvr

# Avatar Creation
1. Open Unity Hub and accept license
2. Open in browser: `unityhub://2022.3.22f1/887be4894c44.`
3. Select Android Build Support (not SDK) and Windows Build Support (Mono)
4. Open Alcom and import the avatar files
5. Add Linux patches `https://befuddledlabs.github.io/LinuxVRChatSDKPatch/`
